# CocosCreator3_SingleHtmlPlayable
CocosCreator 3.0.0-3.0.1 Single Html Playable Demo

CocosCreator 3.0.0-3.0.1 建構單一可玩式HTML  
您可以下載index.html來開啟觀看

目前這自動化轉換為單一可玩式HTML的工具還在發展中  
尚未經過完整的測試除錯  
也還未實作壓縮以及各廣吿渠道  
故暫時不對外公開這自動化轉換工具  
這僅僅只是測試用的展示HTML
